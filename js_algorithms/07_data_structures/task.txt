Задание
Односвязный список — базовая динамическая структура данных в информатике, состоящая из узлов, каждый из которых содержит как собственно данные, так и ссылку на следующий узел списка.

В списках могут встречаться, так называемые, циклы. Это происходит в том случае если последний элемент списка имеет ссылку на один из элементов этого же списка. В обычной ситуации последний элемент не имеет никаких ссылок, так как он последний.

solution.js
Реализуйте и экспортируйте функцию isCircular, которая определяет есть ли циклы в списке.

Интерфейс создания списка:

use function App\Node\make;
use function App\Node\setNext;

// создание узла, $value - данные
$first = make($value);

// создание второго узла, $second будет ссылаться на $first
$second = make($anotherValue, $first);

getNext($second); // $first

Пример:

$last = make(8);
$head = make(7, $last);
$head = make(6, $head);

Solution\isCircular($head); // false
Пример:

$last = make(8);
$head = make(2, $last);
$head = make(1, $head);
setNext($last, $head);

Solution\isCircular($head); // true
Подсказки
Единственная функция, которая вам понадобится это getNext. Эта функция возвращает вам следующей элемент списка.
